# **App Name**: Rhythmic

## Core Features:

- User Authentication: Secure registration, login, and logout functionality using bcrypt and JWT.
- Music Search: Search for music using keywords and filters via an integrated open-source music API.
- Music Player: A music player component with standard controls, progress bar, and volume adjustment.
- Playlist Management: Create, edit, and delete playlists, and manage songs within them, all stored in the MySQL database.
- Personalized Dashboard: User dashboard displaying recently played songs and playlists.
- Genre Recommendations: Suggest songs or artists that a user may like based on their listening habits

## Style Guidelines:

- Primary color: Deep violet (#9400D3) for a modern and sophisticated feel.
- Background color: Light grey (#E6E6E6), subtly violet-tinted, for a calm background.
- Accent color: Electric indigo (#6F00FF) to highlight active elements and interactive components.
- Body font: 'PT Sans', a sans-serif for clean and accessible text.
- Headline font: 'Playfair', a modern serif for elegant headlines. If longer text is anticipated, use this for headlines and 'PT Sans' for body.
- Use sleek, minimalist icons for player controls and playlist actions.
- Responsive layout with clear sections for music discovery, player, and playlists.
